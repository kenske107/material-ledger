⚙ 연결 설정이 필요해요

Google Apps Script 백엔드 URL이 설정되지 않았습니다. 아래 순서대로 설정해주세요.

  1. Google Sheets에서 새 스프레드시트 생성
  2. 메뉴 확장 프로그램 > Apps Script 클릭
  3. 제공된 AppsScript-Code-final.gs 내용을 전체 붙여넣기 후 저장
  4. 배포 > 새 배포 > 웹 앱 선택, 액세스 권한 "Anyone"으로 배포
  5. 발급된 .../exec URL을 복사
  6. 이 HTML 파일을 텍스트 편집기로 열어 const API_URL = "..." 부분에 붙여넣고 저장
